# 🐄 Ganadería Pamora v1.0

Sistema integral de gestión ganadera diseñado para el control de producción, costos, rentabilidad y gestación. Desarrollado con HTML5, CSS3 y JavaScript vanilla, integrado con Google Sheets como base de datos persistente.

## 🚀 Características Principales

- **Panel de Control Dinámico:** Visualización inmediata de KPIs de producción y salud del hato.
- **Registro de Ordeño:** Entrada rápida de datos por animal con sincronización en tiempo real.
- **Análisis de Rentabilidad:** Cálculo dinámico de costos por litro, márgenes de ganancia y análisis histórico.
- **Control de Gestación:** Estimación automática de partos (283 días) y alertas de proximidad (<30 días).
- **Gestión de Costos:** Seguimiento de gastos en concentrados, veterinaria y mantenimiento.
- **Exportación de Reportes:** Generación de informes profesionales en PDF.

## 🛠️ Instalación y Uso

1. **Clonar el repositorio:**
   ```bash
   git clone https://github.com/doccpaez4-commits/Programa-Ganaderia.git
   ```
2. **Abrir localmente:** Simplemente abre `index.html` en cualquier navegador moderno.
3. **Configuración Backend:** El sistema utiliza Google Apps Script. Asegúrate de configurar tu `APPS_SCRIPT_URL` en `app.js`.
